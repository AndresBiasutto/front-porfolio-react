# Front Portfolio React

Una aplicación creada para conocernos mejor.

## Instrucciones para ejecutar localmente

1. Clona este repositorio.
2. Abre la carpeta del proyecto en Visual Studio Code.
3. En la terminal, ejecuta el comando `npm install` para instalar todas las dependencias.
4. Luego, ejecuta el comando `npm run dev` para iniciar el servidor.
5. Abre tu navegador y ve a `http://localhost:5173` para ver la aplicación en funcionamiento.
